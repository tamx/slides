## Node.js とは？

- JavaScript の実行環境
- ブラウザがなくても、 JavaScript コードを実行できる。
- Web サーバーなどを JavaScript で作成できる。

---
## 動かしてみよう。

test.js
~~~
console.log("Hello, World!");
~~~

実行
~~~
% node ./test.js
~~~
